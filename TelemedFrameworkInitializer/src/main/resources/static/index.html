Telemedizin Framework Initalizer

Willkommen beim Telemedizin Initializer!

Dieser Initializer bietet Ihnen fertige Module und somit einen guten für Ihre Telemedizin Applikation

Wählen Sie aus den angegebenen Modulen starten Sie in wenigen Schritten Ihre eigene Telemedizin-Applikation!

Genauere Informationen zum Starten entnehmen Sie bitte der Dokumentation

Communication Module

Das Communcation-Modul ermöglicht Ihnen die Kommunikation mit Ihren Patienten per Videocall

Symptom Tracking Module

Mit dem Syptom Tracking-Modul können Ihre Patienten Iher Symptome übersichtlich verwalten

Termine Modul

Mit dem Termin-Modul können Sie mit Ihren Patienten einfach online Termine ausmachen

Kurs Modul

Mit den Kurs-Modul können Sie Information und Material bereitstellen, welches Ihre Patienten abonnieren können

Schnittstellen Module

Das Schnittstellen-Modul ermöglicht Ihnen den Export der Patientendaten per FHIR-Schnittstelle zur Kommunikation mit anderen eHealth-Applikation
